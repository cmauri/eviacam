Modos de premer (facer clic)

Clic por demora

Este é o método predeterminado do aplicativo para xerar clics. Para realizar o clic, deter o punteiro uns instantes na área onde desexe premer. Despois do tempo preestabelecido, producirase automaticamente o clic. A acción predeterminada a realizar é o clic esquerdo. Utilice a xanela do clic para seleccionar unha acción diferente. Ver Xanela do clic.

Se a opción de alertas visuais está activada, aparecerá unha barra de progreso á beira do punteiro que amosará o contador. Non dispoñíbel para Windows.

Clic por xestos

Este método de xeración de clic permítelle ao usuario seleccionar a acción a realizar movendo sómente a cabeza. Para xerar un clic, deter o punteiro na área onde desexe premer. Despois do tempo preestabelecido, mover a cabeza cara a dirección asociada á acción desexada. Cando teña rematado o tempo de demora do xesto producirase o clic.

Pode modificar as accións asociadas a cada movemento a través de: Configuración -> Opcións -> Clic

Se a opción de alertas visuais está activada, aparecerá unha barra de progreso á beira do punteiro para indicar os tempos de demora. Cando vostede mova a cabeza para indicar a acción que desexa realizar, aparecerá unha liña que seguirá ao punteiro desde a área onde se premerá (realizará o clic).

NOTA: Recomendamos activar a opción de alertas visuais para traballar con este método de xeración de clics.
